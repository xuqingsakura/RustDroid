# RustDroid IDE — 完整页面原型设计说明

## 整体设计

### 主题
- **深色模式**：Tokyo Night 配色（背景 `#1a1b26`，文字 `#c0caf5`，强调 `#7aa2f7`）
- **浅色模式**：Tokyo Night Light 变体（背景 `#e6e7ef`，文字 `#343b58`，强调 `#34548a`）
- 通过 `data-theme="light"` 属性切换，右上角太阳/月亮按钮控制

### 图标系统
- 全部使用 SVG 矢量图标，通过 CSS `mask-image` + `data:image/svg+xml` 实现
- 颜色跟随 `currentColor`，支持主题切换
- 共 22 种图标定义（search/git/play/bug/bot/settings/file/folder/code/home/plus/terminal/alert/info/send/download/chevron-right/chevron-down/sun/moon/refresh/device/check/close）

### 字体
- UI：系统字体栈（-apple-system, Segoe UI, system-ui）
- 代码：Consolas / SF Mono 等宽字体

## 各功能区详细说明

### 1. 启动页
- 居中布局，标题 RustDroid + 副标题
- 3 个操作卡片：新建项目 / 打开项目 / 克隆仓库
- 最近项目列表：项目名 + 路径 + 时间，点击直接进入编辑器

### 2. 新建项目向导
- 模态对话框，680px 宽
- 三步进度指示（选择模板 → 项目配置 → 确认）
- 3 个模板卡片（Empty Activity / Compose / Navigation）
- 表单字段：项目名称、包名、保存位置、最低 SDK、语言、构建工具
- 底部取消/创建按钮

### 3. 编辑器（核心界面）
- CSS Grid 四栏布局：菜单栏 + 活动栏(48px) + 侧边栏(260px) + 编辑器 + 右侧面板(300px)
- 活动栏 7 个图标按钮，点击跳转对应功能区
- 文件树：缩进式展开/折叠，当前文件高亮
- 编辑器标签页：多文件切换，关闭按钮
- 代码区：行号 + 语法高亮（关键字紫/类型青/字符串绿/注释灰）
- 右侧面板：大纲（结构树）+ 属性面板
- 底部面板：3 个标签页可切换
  - 终端：命令行输出
  - 问题：错误/警告列表，带行号定位
  - 输出：LSP/Gradle 状态信息
- 状态栏：Git 分支、问题数、编码、语言、设备

### 4. 搜索
- 搜索输入框 + 替换输入框
- 搜索选项：区分大小写、正则、全词匹配、文件类型过滤
- 结果统计："4 个文件中找到 6 个结果"
- 按文件分组的结果列表，可折叠/展开
- 匹配文本高亮显示

### 5. 版本控制
- 左侧栏：分支选择、暂存更改(2)、更改(3)、最近提交
- 每个文件显示状态标记：M(修改)/A(新增)/D(删除)/U(冲突)
- 主区域顶部：提交信息输入 + 提交/推送按钮
- 差异对比视图：行号对照、添加行(绿)/删除行(红)/上下文行(灰)

### 6. 运行
- 顶部配置栏：运行配置下拉 + 设备选择下拉 + 操作按钮（运行/调试/构建APK/清理）
- 四面板网格布局：
  - 构建任务：每个 Gradle task 耗时，构建成功总结
  - 构建日志：带颜色级别的日志输出
  - 已连接设备：设备名/API/连接方式/在线状态
  - Gradle 任务列表：8 个常用任务可点击执行

### 7. 预览
- 三栏布局：设备/文件列表(220px) + 预览画布 + 属性面板(260px)
- 工具栏：设计/代码/分屏视图切换、竖屏/横屏、缩放比例、刷新
- 手机模拟器框架：刘海、状态栏、应用内容区
- 模拟应用界面：AppBar + 4 个数据卡片 + 浮动按钮
- 属性面板：布局属性、主题颜色、渲染引擎信息

### 8. 调试
- 顶部调试工具栏：继续/步过/步入/步出/停止 + 会话信息
- 四面板布局：
  - 变量：当前作用域变量，树形缩进，显示类型和值
  - 调用栈：函数名+文件:行号，当前帧高亮
  - 断点：文件:行号列表，启用/禁用状态
  - 调试控制台：eval 求值、命令执行

### 9. 设置中心
- 左侧导航(220px)：6 个子页面
- 右侧滚动内容区，每个设置项为"标签+描述"与"控件"的左右布局

| 子页面 | 内容 |
|--------|------|
| 编辑器 | 字体、字号、行高、连字、自动保存、自动格式化、代码补全、行号、缩进 |
| 外观 | 深色模式、主题配色(4种)、侧边栏位置、活动栏/状态栏可见、界面缩放 |
| SDK 管理 | SDK 路径、已安装平台列表(API 24-35)、构建工具/平台工具/NDK |
| 构建配置 | Gradle 版本、AGP 版本、离线模式、并行构建、构建缓存、JDK 路径/版本 |
| 版本控制 | 用户名、邮箱、默认分支名、自动暂存、GPG 签名、pre-commit 钩子 |
| 快捷键 | 方案预设(RustDroid/VSCode/IntelliJ/Eclipse)、15 个常用快捷键列表 |

### 10. 关于
- 居中布局，大标题 RustDroid
- 版本号 + 开源许可
- 4 个链接：源代码/文档/更新/反馈
- 技术栈网格：Tauri/React/Rust/Monaco/tree-sitter/ropey/jdtls/Kotlin LSP
- 商标声明和版权信息

## 交互功能

| 功能 | 触发方式 | 效果 |
|------|----------|------|
| 视图切换 | 点击顶部导航标签 | 切换到对应功能区 |
| 主题切换 | 右上角太阳/月亮按钮 | 深色↔浅色模式 |
| 底部面板标签 | 编辑器内点击标签 | 切换终端/问题/输出 |
| 设置子页面 | 设置左侧导航点击 | 切换设置分类 |
| 折叠面板 | 点击面板标题 | 展开/折叠内容 |
| 搜索结果折叠 | 点击文件分组标题 | 展开/折叠该文件的结果 |
| Git 分组折叠 | 点击 Git 分区标题 | 展开/折叠该分区的文件列表 |
| 开关切换 | 点击 toggle 控件 | 切换开关状态 |
| 模板选择 | 点击模板卡片 | 选中高亮 |
